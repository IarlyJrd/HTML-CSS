Tags Em HTML

Em html podemos usar os seguintes elementos da lista. Passe o mouse em cada um dos elementos e veja qual tag devera utilizar

    Paragrafos
    use a tag 'P'
    Titulos
    Use a tag 'h1'
    imagens
    Use a tag 'img'
    Abreviações
    use a tag 'abbr'
    Codigos fonte
    use a tag 'fonte'
    Listas Numeradas
    use a tag 'ol'
    Lista com demarcadores
    use a tag 'ul'
